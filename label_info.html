Welcome to the Labeling Part

In this part, you will select each sketch object appear in the scene and write their object categories.

Instructions:

  • You will be asked to label each sketch object that appear in the given scene.
  • Click to SEARCH CATEGORY button to find the correct object category.
  • Click to SUBMIT button if you find the correct category name.
  • Labelled object strokes become GREEN.
  • You can label objects in any order you want.
  • When the scene becomes all GREEN, you can continue with the next scene.
