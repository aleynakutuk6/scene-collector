Welcome to the Labeling Part

In this part, you will select each sketch object appear in the scene and write their object categories.

Instructions:

  • You will asked to label every sketch objects appear in the scene.
  • Write the corresponding category name in a text box given on the left of the scene.
  • Once you write the correct category name, click SUBMIT and strokes will turn into GREEN color indicating that you have done with labeling of those strokes.
  • Label all sketch object one by one.
  • You can start labeling from any object appear in the scene.
