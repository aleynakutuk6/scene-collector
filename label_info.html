Welcome to the Labeling Part

In this part, you are going to label each object that appears on the scene.

Please carefully read each set of instructions. You'll be given a series of questions that you must answer correctly to continue with the next page. Failure to answer all questions correctly will result in redirection to the information page.

Instructions:

  • You are asked to assign a label to each object that appears in the provided scene.

  • You will find the correct object category from a given list.

  • You may write the category name to the given search area to filter the given categories.

  • When you find the correct category name, click the SUBMIT button.

  • If the correct object category does not appear in the given list, you should write the external category name and submit.

  • Once you have labelled an object instance, the object will change its color.

  • You are allowed to label objects in any order you want.

  • When the scene becomes all colorful, you can continue with the next scene.
