Welcome to the Scene Sketch Dataset Collection Application

This is an example data collection application built for researchers who need a sketch scene data collection interface.

Instructions:

  • You will be given with a scene description and a drawing canvas on the next page.
  • You are asked to draw a scene by referring to the description.
  • While drawing the scene, explain it to the instructor verbally.
  • You may draw additional objects which is not mentioned in the description.
  • Imagine that the instructor does not know about the scene description, only can see your drawing canvas.

Note:

Your scene drawings will be used in a research project by a graduate student. If you agree to give a permission, you can write your name and surname then click to the START button.

**This dataset will be used by M.Sc. student Aleyna Kutuk.
